# Instructions

- The first render would only load the shapes.
- Clicking a polygon to see the metadata.
- Selecting one or more items from the `filter classes` section to filter the polygons and get the scalar data for the sepecific classes.
- Use the `year` and `day` slider to update the polygon colors.
